# SONAR Rock vs Mine Prediction using Machine Learning

## Overview
This project predicts whether an object is a Rock or a Mine using SONAR signal data and Machine Learning techniques.

## Problem Statement
The goal is to classify sonar signals reflected from objects as either:
- Rock
- Mine
