# Diabetes Prediction using Machine Learning

## Overview
This project predicts whether a person is diabetic or non-diabetic using Machine Learning techniques and medical diagnostic data.

---

## Problem Statement
Diabetes is one of the most common chronic diseases worldwide. Early prediction can help in proper treatment and prevention.

The goal of this project is to build a Machine Learning model that predicts diabetes based on medical parameters.

- Glucose Level
- Blood Pressure
